#1. CSS SYNTAX

1.1 Cambia el color de todos los elementos <p> a "rojo".

            <style>
                p {color: red;}
            </style>

1.2 Cambie el color de todos los elementos <p> y <h1> a "rojo". Agrupa los selectores para minimizar el código.

            <style>
                h1,p {color: red;}
            </style>


#2. CSS HOW TO...

2.1 Agregue una hoja de estilo externa con la URL: "mystyle.css"..

            <head>
              <link rel="stylesheet" type="text/css" href="mystyle.css">
            </head>

2.2 Establezca "background-color: linen" para la página, usando una hoja de estilo interna.

            <head>
              <style>
              body {background-color: linen;}
              </style>
            </head>


#3 CSS BACKGROUND

3.1 Establezca el color de fondo de la página en "linen" y el color de fondo de <h1> en "lightblue".

              h1 {background-color: lightblue;}
            </style>

3.2 Establezca "paper.gif" como la imagen de fondo de la página.

            <style>
              body {background-image: url("paper.gif");}
            </style>


#4 CSS BORDER

#12 CSS LINKS

            <style>
              a { color: green; }
            </style>

12.2 Establezca el color de los enlaces no visitados en "rojo" y el color de los enlaces visitados "azul".

              /* unvisited link */
              a:link { color: red; }
              
              /* visited link */
              a:visited { color: blue; }
              
              /* mouse over link */
              a:hover { color: black; }
              
              /* selected link */
              a:active { color: green; }
              </style>

#20 CSS PSEUDO - CLASSES

#35 CSS ANIMATIONS

35.1 Agrega una animación de 2 segundos para el elemento <div>, que cambia el color de rojo a azul. Llama a la animación "example"

            <style> 
              div { width: 100px; height: 100px; background-color: red;
                    animation-name: example; animation-duration: 2s; }  
              @keyframes example {
                from {background-color: red;}
                to {background-color: blue;} }
            </style>

35.2 Agregue los siguientes 5 pasos a la animación "example" (utilizando 0%, 25%, 50%, 75% y 100%):

0% - Ajuste el color de fondo a "rojo", la posición izquierda a "0px", la posición superior a: "0px"

25% - Ajuste el color de fondo a "azul", la posición izquierda a "0px", la posición superior a: "200px"

50% - Establezca el color de fondo en "verde", la posición izquierda en "200px", la posición superior en: "200px"

75% - Ajuste el color de fondo a "amarillo", posición izquierda a "200px", posición superior a: "0px"

100% - Establezca el color de fondo en "rojo", la posición izquierda en "0px", la posición superior en: "0px"

            <style> 
              div { width: 100px; height: 100px; position: relative; 
                    background-color: red; animation-name: example; animation-duration: 4s; } 
              @keyframes example {
                0%   {background-color: red; left:0px; top:0px;}
                25%  {background-color: blue; left:0px; top:200px;}
                50%  {background-color: green; left:200px; top:200px;}
                75%  {background-color: yellow; left:200px; top:0px;}
                100% {background-color: red; left:0px; top:0px;}
              }
            </style>
